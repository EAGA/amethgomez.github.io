Experiencias

Ene 2020 - Sep 2020 Gerente de Tecnología en Playa Blanca Resort

Es de destacar que como Gerente Senior de Tecnología para el Hotel Playa Blanca durante mi gestión se propuso la adquisición, implementación y puesta en marcha de un sistema ERP, esta herramienta solventaría los inconvenientes de no tener una plataforma unica para los procesos contables, esta plataforma se consideró adquirirla tipo SAAS.

Al mismo tiempo el departamento de RRHH presentaba una necesidad de un HR o Human Resources de este modo se realizó la evaluación para adquirir dicho programa con el objetivo de optimizar las tareas del departamento este proyecto también consideraba ser una plataforma tipo SAAS.

Dic 2018 - Oct 2019 Gerente de tecnología Mi Financiera

En esta oportunidad se me permitió, asumir las funciones de Gerente de Tecnología para Mi Financiera.

Como Gerente de Tecnología se presentaron muchos retos que la empresa mantenía: Renovación tecnológica en la empresa producto de la obsolescencia tecnológica esto inicio el proyecto con el que dotamos de equipos con tecnología actualizada.

Implementación de centro de datos local con lo cual se desarrollaron distintos servidores para las tareas de los diversos departamentos aquí se implemento tecnología de virtualización.

Implementación de Sistema ERP para las distintas funciones y tareas ciclo contables y demás.

Implementación de sitio de respaldo este proyecto incluyó el desarrollo de un DRP

Implementación de Sistema de Finanzas SIAFI

Implementación de Red local en casa matriz, ademas se definió la topologia de red entre sucursales para la comunicación en tiempo real con este trabajo se dotan de servicios de seguridad (firewall) y conexión remota vía VPN pudiendo desarrollar el trabajo remoto.

Diseño, desarrollo e implementación de políticas o manuales de tecnología dotando a la empresa de un conjunto de buenas practicas.
