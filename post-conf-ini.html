Configuración Inicial

01-05-2021

Hay algo muy claro para gustos los colores y tecnología no escapa a esta diferencia tal vez algunos no concuerden con mi configuracion inicial pero recuerden este es una guia y muy enfocada con mis experiencias y practicas:

  • Sistema Operativo
  • Programas Iniciales

Sistema Operativo

Aunque sea muy predecible si Windows es el sistema operativo el que utilizo por defecto para la mayoría de mis ensayos o practicas la pregunta porque y podrán haber muchas respuestas pero la principal sigue siendo el sistema operativo mas utilizado a nivel comercial sin dejar a un lado android que compite fuertemente pero podrás coincidir que si vas a una oficina publica o privada encontraras computadores windows dicho esto común que utilice maquinas virtuales

Pros

  • Son fáciles de configurar
  • Son fáciles de Clonar
  • Sencillo para crear snapshot
  • No Afecto la configuración de la maquina principal

Contras

  • Se necesita espacio en disco
  • Se necesita memoria ram
  • Se necesita procesador

seguro me dirás entonces que haces con las maquinas virtuales después de no usarlas fácil se convierten en OVA que es un archivo con un peso menor que la maquina virtual original y que contiene todo el conjunto hasta el momento de su creación. Si bien es cierto existen otras alternativas como Docker y Kubernetes los cuales hablaremos en otro momento. Repito es la practica que me ha resultado en mi experiencia. No olvidemos instalar todas las actualizaciones del sistema operativo.

Programas Iniciales

En cuanto a programas iniciales hay un abanico de opciones para todos los casos y procedemos a enumerarlos y definir sus usos:

  • Adobe Reader: Siempre es efectivo un lector PDF local no en el explorador
  • Chrome: Aunque consume mucha memoria ram es mi explorador de preferencia
  • 7zip: Excelentes resultados para compresión de archivos inclusive lo utilizo como herramienta durante la ejecución de scripts o tareas programadas
  • Notepad++: aunque tiene otras funciones lo utilizo como libreta de apuntes
  • Visual Studio Code: se ha convertido en mi IDE de programación por lo general recomiendo las extensiones: Markdown Preview Enhanced, Andromeda y palenight
  • Git: como control de versiones aquí te recomiendo primero instalar tu IDE de programación primero y git posterior a la instalación ejecutar via power shell los siguientes comandos git config --global user.name "Tu nombre" git config --global user.email tunombre@correo.com

Con estos pasos tendremos la maquina inicial los siguientes programas dependerán de cada objetivo especifico, se que muchos me podrán decir tienes muchas maquinas virtuales pero detrás existe un control de que ningún programa entrará en conflicto con otro espero te sirva de ejemplo para tus practicas

Recuerda utilizar la version mas reciente y estable y mantente siempre actualizado
