Instalacion y Programacion en R con RStudio

29-04-2021

En este apartado desarrollaré los siguientes puntos:

  • Definicion y Conceptos
  • Instalacion de R y RStudio
  • Entorno de Trabajo
  • Codigos de Ejemplo

Definicion y Conceptos

Antes de iniciar primero debemos de diferenciar R de RStudio. De este modo R es un lenguaje de programación cuyo objetivo es el análisis estadístico y por otro lado RStudio es el IDE o entorno de desarrollo integrado para la programacion en R.

Instalacion de R y RStudio

Como ya tenemos los conceptos iniciales procedamos a la instalacion, la instalacion para este escenario la realizare sobre un sistema operativo windows 10 primero instalaremos R el cual lo obtenemos desde la pagina https://cran.r-project.org/ seleccionamos el sistema operativo con el deseamos trabajar, descargamos y procedemos a la instalamos

Ahora procedemos a instalar RStudio el cual lo obtenemos desde la pagina https://www.rstudio.com/products/rstudio/download/ una vez mas seleccionamos el sistema operativo con el deseamos trabajar, descargamos y procedemos a la instalamos

Es importante recordar utilizar la version mas reciente y estable

Entorno de Trabajo

Es momento de identificar nuestro ambiente de trabajo como todo IDE posee barra de herramientas para realizar las funciones basica de abrir, guardar,vistas, etc. seguido veremos 4 panel los cuales describiremos acontinuacion:

  • Panel Script:

    • Es el dedicado para la escritura de nuestros scripts o codigos de programacion.

  • Panel Consola.

    • Este panel nos presenta los resultados de todas aquellas ejecuciones o instrucciones dadas en nuestros scripts.

  • Panel de Entorno de Trabajo.

    • En la medida que ejecutemos script o codigos de programas estos se definiran en nuestro entorno de trabajo.

  • Panel de Ejecucion.

    • Todo script ejecutado y cuyo resultado se obtenga en la consola puede tener una representacion grafica en este panel asi que por definicion en este panel veremos nuestros graficos. Sin embargo tambien funciona para realizar tareas de actualizacion de paquetes.
